## 📌 Project Description
This project implements a wireless drone control system using a joystick interface and the NRF24L01 transceiver module. Joystick input values are transmitted in real time to the drone, enabling responsive and low-latency flight control.

---

## 🛠 Hardware Used
- Microcontroller (Arduino / STM32)
- NRF24L01 2.4GHz RF Module
- Joystick Module
- Drone Flight Controller
- Power Supply & Wiring

---

## 💻 Software & Tools
- Arduino IDE / STM32CubeIDE
- Embedded C
- RF24 Library
- Serial Debugging Tools

---

## ⚙ Key Features
- Real-time wireless joystick control
- Low-latency RF communication
- Stable 2.4GHz control link
- Modular and scalable design
